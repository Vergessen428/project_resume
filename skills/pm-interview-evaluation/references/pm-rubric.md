# PM Rubric

The six skills remain stable. Each skill has four observable dimensions; weights sum to 100.

| Skill ID | Dimensions |
| --- | --- |
| `product_sense` | `user_problem` 30, `goal_definition` 20, `tradeoff` 30, `prioritization` 20 |
| `story_ownership` | `scope` 30, `decision` 30, `collaboration` 15, `result_learning` 25 |
| `metrics_experiment` | `definition` 25, `decomposition` 20, `attribution` 30, `experiment_quantify` 25 |
| `execution_collaboration` | `planning` 25, `alignment` 25, `resource_tradeoff` 20, `closure` 30 |
| `structured_communication` | `structure` 30, `directness` 20, `precision` 25, `probe_response` 25 |
| `business_context` | `jd_link` 30, `user_business` 25, `market_context` 20, `role_fit` 25 |

## Score Anchors

- `1`: no observable behavior, or the answer is materially mismatched.
- `2`: relevant point is mentioned, but a key definition, action, trade-off, or result is missing.
- `3`: basically complete example with one important gap.
- `4`: specific and reasoned behavior with only a small detail gap.
- `5`: evidence closes the loop across user/business context, decision, metric, result, and learning.

Interpolate 2 and 4 only when the transcript supports the intermediate behavior. A missing dimension is not a default 3.
