# 秋招助手 · 静态展示版

这是「秋招助手」的**纯前端只读演示**，无需后端即可运行，用于展示“诊断总结 → 短期训练 → 长期记忆”的完整路径。

- 面试复盘：展示训练分、证据覆盖度、六项 PM 能力、子维度评分、缺口、追问和训练验收
- 阶段成长报告：展示跨场重复缺口、能力变化、长期训练和数据边界；单场短期记忆单独呈现
- 面经资料库：展示 Agent 查询轨迹、公开页自动读取尝试、相关度拆解和来源状态
- 公开面经候选是合成演示数据，明确标为待确认，不对应可验证原帖；数据在 `demo_data.js`
- 公开候选是合成演示数据，明确标为待确认，不对应可验证原帖；真实后端会尝试读取限定平台公开 HTML，但不会登录、读 Cookie 或绕过访问控制
- 搜索、保存、AI 复盘、上传等写操作在静态版中会提示「需在完整版体验」

完整版（带实时 AI 生成、数据存储）见项目主仓库的 `app/`，可本地运行或部署到带后端的平台。
