# 秋招助手 · 静态展示版

这是「秋招助手」的**纯前端只读演示**，无需后端即可运行，用于给他人展示产品的完整界面
与一份带原文证据的深度示例复盘。

- 面试复盘：展示带原文证据的结构化复盘、PM 能力诊断、逐题复盘、行动项
- 简历库 / 面经资料库 / 阶段成长报告：均有预置示例；搜索相关度和 PM 评分锚点也有展示
- 公开面经候选是合成演示数据，明确标为待确认，不对应可验证原帖；数据在 `demo_data.js`
- 公开搜索按钮展示查询轨迹与来源状态；保存、AI 复盘、上传等写操作在静态版中会提示「需在完整版体验」

完整版（带实时 AI 生成、数据存储）见项目主仓库的 `app/`，可本地运行或部署到带后端的平台。
